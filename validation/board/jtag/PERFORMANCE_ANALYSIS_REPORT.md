# SOCS HLS 板级验证性能分析报告

**日期**: 2026-05-06

---

## 1. HLS IP 性能

- **时钟频率**: 300 MHz (目标), 457 MHz (估计)
- **计算延迟**: 2,636,242 cycles
- **计算时间**: 8.79 ms

## 2. CPU 参考实现

- **计算时间**: 49.6 ms

## 3. 加速比

- **纯计算加速比**: **5.6x**

## 4. JTAG 传输开销

- **传输时间**: ~35 分钟
- **计算时间**: 8.79 ms
- **传输/计算比**: 238977x

## 5. 关键发现

✅ **HLS IP 功能正确** (RMSE = 2.93e-08)

✅ **计算性能优秀** (5.6x 加速)

⚠️ **JTAG 传输是主要瓶颈** (占比 >99.9%)

## 6. 优化建议

1. **使用 PCIe 或以太网替代 JTAG**
   - PCIe Gen3 x8: ~6 GB/s
   - 10GbE: ~1 GB/s
   - JTAG: ~1 MB/s

2. **批量处理多个图像**
   - 一次传输，多次计算
   - 分摊传输开销

3. **数据预加载到 DDR**
   - 避免每次都传输输入数据
   - 只传输变化的参数
