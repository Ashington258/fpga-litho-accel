## Mask分辨率 → HLS参数对照表

### 核心公式

$$N_x = \left\lfloor \frac{NA \times L_x \times (1 + \sigma_{outer})}{\lambda} \right\rfloor$$

### 参数派生关系

```
Nx (频域采样点数)
  │
  ├──→ kerX = 2×Nx + 1  (Kernel宽度)
  │     例: Nx=4 → kerX=9
  │
  ├──→ convX = 4×Nx + 1  (卷积输出宽度)
  │     例: Nx=4 → convX=17
  │
  └──→ fftConvX = next_pow2(convX)  (FFT执行尺寸)
        例: convX=17 → fftConvX=32
        例: convX=65 → fftConvX=128
```

### FFT尺寸跃变点

| convX范围 | next_pow2结果 | FFT尺寸 |
| --------- | ------------- | ------- |
| 1-2       | 2             | 2×2     |
| 3-4       | 4             | 4×4     |
| 5-8       | 8             | 8×8     |
| 9-16      | 16            | 16×16   |
| 17-32     | 32            | 32×32   |
| 33-64     | 64            | 64×64   |
| 65-128    | 128           | 128×128 |

### 使用建议

**如果要修改mask分辨率**：

| 新分辨率  | 需要的操作                                       |
| --------- | ------------------------------------------------ |
| 256×256   | 需要新综合 Nx=2 的 HLS IP                        |
| 1024×1024 | 需要新综合 Nx=8 的 HLS IP                        |
| 2048×2048 | 推荐：综合 Nx=16 最大IP，小分辨率通过padding适配 |
