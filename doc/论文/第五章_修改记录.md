### 4. C++ CPU基准的多线程说明（5.4.3节）

**修改位置**: 第263行、第277-279行、第285行

**修改前**:
```
C++ 参考实现中，10 核 SOCS 在线成像（Module 7，纯计算，不含数据加载与文件 I/O）
实测耗时 \(T_{\text{SOCS,CPU}} = 35.6\text{ ms}\)。加速比为：

| 完整 TCC 直接成像   | C++ (单精度, -O2) | 45.176 ms | 10.57 ms             | **4.28×** |
| 10 核 SOCS 在线成像 | C++ (单精度, -O2) | 35.6 ms   | 10.57 ms             | **3.37×** |
```

**修改后**:
```
C++ 参考实现中，10 核 SOCS 在线成像（Module 7，纯计算，不含数据加载与文件 I/O，单线程执行）
实测耗时 \(T_{\text{SOCS,CPU}} = 35.6\text{ ms}\)。加速比为：

| 完整 TCC 直接成像   | C++ (单精度, -O2, 单线程) | 45.176 ms | 10.57 ms             | **4.28×** |
| 10 核 SOCS 在线成像 | C++ (单精度, -O2, 单线程) | 35.6 ms   | 10.57 ms             | **3.37×** |
```

**修改理由**:
- 明确C++实现为单线程执行
- 避免审稿人质疑48核CPU是否启用多线程
- 增强加速比3.37×的说服力
